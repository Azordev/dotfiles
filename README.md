# The Project

These are my dotfiles. Take anything you want, but at your own risk. It targets debian systems, but 
it should work on *nix as well (with apt-get).

# Features

- Aliases
- Prompt theme
- `bash` functions
- `vscode` settings

# Prerequisites

- Debian 8+ or derivatives. It should work on *nix as well
- `git`
- `vscode`
